# Fagbegrep MFL2
Flashcard-verktøy for elever i Markedsføring og ledelse 2 (SAM3046) ved Bergen Private Gymnas.

## Innhold
Rundt 200 fagbegreper fra _Visjon 2_, organisert i 14 tematiske kategorier. Elevene kan øve på alle begreper i tilfeldig rekkefølge, eller velge ett eller flere temaer.

## Bruk
Åpne lenken i nettleseren. Fungerer på mobil, nettbrett og PC.

- **Trykk** på kortet for å se forklaringen
- **Sveip** eller bruk piltastene for å bla
- **Mellomrom** snur kortet
- **Stokk om** randomiserer rekkefølgen

## Teknisk
Én enkelt `index.html`-fil uten avhengigheter. Hostes via GitHub Pages.
